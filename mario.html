CARD FLIP

LET'S A-GO!

Get ready for a gut punch of nostalgia in this take on the classic Super Mario 3 card match game! Featuring updated hand drawn graphics, it's 16 bit realness for the modern age. Beat the clock and flex those memory muscles!

TECHNOLOGY USED

HTML | CSS | JavaScript

FEATURES

Created entirely in vanilla JavaScript using DOM manipulation, this is a faithful recreation of the Mario classic complete with card animations, randome card order generation, card matching, game timer and score cards.
